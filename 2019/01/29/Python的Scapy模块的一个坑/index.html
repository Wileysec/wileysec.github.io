Python的Scapy模块的一个坑

本文将讲述Python的Scapy模块的一个大坑

导入Scapy模块错误

错误提示：ImportError: cannot import name 'NPCAP_PATH'

不能导入名字为NPCAP_PATH的方法

这个解决方法一直找不到。

scapy模块从2015年的2.0版开始创建了一个独立的scapy分支，旨在仅支持python3（scapy3k）。它在某些发行版中被打包为python3 scapy，在pypi上被打包为scapy-python3，这会导致用户之间的混淆。很快就不应该这样了。从2.4.0开始，scapy除了支持python2外，还支持python3。scapy 2.4.0版应被视为官方的scapy代码库，已更名为kamene。

当我们使用kamene模块去执行脚本的时候就不会出现此情况了。

所以建议大家使用Python去实现ARP的时候还是使用kamene吧。
