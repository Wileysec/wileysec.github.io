学习Linux碰到的一些坑

本文讲述的是在学习Linux当中踩过的一些坑，记录学习。

普通用户无法使用ifconfig命令

如果没有安装net-tools工具，先安装net-tools。

安装net-tools命令:apt-get install net-tools

提示:-bash:ifconfig: command not found

是因为没有添加环境，很少遇到这种情况。

运行 export PATH="/usr/local/bin:/usr/bin:/bin:/usr/local/sbin:/usr/sbin:/sbin" 命令。

再运行ifconfig命令即可使用。

Kali Linux安装完搜狗输入法后无法激活输入法

真的，这个问题困扰了我很久，发现怎么装都装不上搜狗输入法，这是什么鬼？

打开fcitx后，出现提示可能是fcitx-configtool没有安装，不过安装的不是这个包，这个包名是：fcitx-config-gtk

执行apt-get install fcitx-config-gtk安装完这个包后，再打开fcitx打开输入法配置，添加搜狗输入法，成功激活搜狗输入法，可以正常使用了。

调整图标大小

按住Ctrl键，滚动鼠标滚轮调整。

Kali Linux更新源问题

在我安装完Kali Linux之后，更新源的时候出现了问题，百度后解决。

提示从存储库更新是不安全的。

输入命令 wget -q -O - https://archive.kali.org/archive-key.asc | apt-key add

可以看到是报了404错误，而且也提示了这个错误。

这里只需要换个更新源就可以了。我换了个清华大学源就解决了。
